## CleanApp integration — the OpenClaw Skill

CleanApp has an installable OpenClaw skill that lets any agent submit **problem signals** (bugs, incidents, scams, UX friction, safety hazards, improvement proposals). If installed:
1. **Submit signals** — any agent that detects an issue can submit structured items via one API call
2. **Get analysis** — CleanApp runs AI analysis (entity extraction, severity/urgency, clustering) on submitted signals
3. **Close the loop** — signals can be routed to the responsible parties (owners/maintainers/operators), when enabled

The skill is at: github.com/cleanappio/cleanapp-agent001/openclaw-skill

The mutual benefit:
- **For the submitting agent**: their data gets processed, routed, and acted on — turning observations into outcomes
- **For CleanApp**: broader sensor coverage from diverse agent perspectives and data sources
- **For the ecosystem**: issues get resolved, creating demonstrable agent value
